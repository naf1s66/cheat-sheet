CSS Colors

Key Value Description
Color Blue There are 140 color codes which are preset. They can be written in english and used directly.
rgb(0, 0, 255) Red, Green and Blue with values between 0 and 255.
rgba (0, 0, 255, 0.5) Same as rgb but includes opacity
hsl (240, 50%, 80%) Hue between 0-360, saturation (0-100%) followed by lightness(0-100%)
hcla (240, 50%, 80%, 0.5) Same as hcl but includes opacity
